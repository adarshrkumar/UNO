# UNO

A web-based UNO card game for two players.

## Features

- Two-player local gameplay
- Card drawing from deck
- Current card display with color and symbol
- Deck management for each player
- Shuffle functionality

## How to Play

1. Click "Start Game" to begin
2. Players take turns drawing and playing cards
3. Match the current card by color or number
4. Use action cards strategically
5. First player to empty their hand wins

## Tech Stack

- HTML/CSS/JavaScript
- Hosted via GitHub Pages
